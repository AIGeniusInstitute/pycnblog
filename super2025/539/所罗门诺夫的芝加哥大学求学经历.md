### 5.2 源代码详细实现

以下是一个Python代码实例，用于计算信息熵：

```python
import math

def entropy(p):
    return -sum(p[i] * math.log2(p[i]) for i in range(len(p)))

# 示例：计算二进制信号的熵

print("Entropy:", entropy(probabilities))
```
