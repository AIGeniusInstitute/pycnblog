#### 5.2 源代码详细实现

以下是一个简单的整数分解算法的Python实现：

```python
from sympy import divisors

def prime_factors(n):
    return list(divisors(n))

# 测试
n = 28
factors = prime_factors(n)
print(f"The prime factors of {n} are: {factors}")
```

#### 5.3 代码解读与分析

这段代码使用了Sympy库中的`divisors`函数来找到给定整数的所有因数。然后，我们将这些因数转换为列表并返回。这个简单的例子演示了如何将数学问题形式化为代码，并通过算法来求解。
