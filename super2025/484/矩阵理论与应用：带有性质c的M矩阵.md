## 2. 核心概念与联系

### 2.1 M-矩阵定义

M-矩阵，也称为非负矩阵，是指满足以下条件的方阵：

* 所有非对角元素均为非正数，即 $a_{ij} \le 0$，其中 $i \neq j$。
* 矩阵的主对角元素均为正数，即 $a_{ii} > 0$。
* 矩阵的逆矩阵存在，且其所有元素均为非负数。

### 2.2 带有“性质c”的M-矩阵定义

带有“性质c”的M-矩阵，是指满足以下条件的M-矩阵：

* 矩阵的非对角元素满足一定的约束条件，例如：
    * 对于任意两个不同的行 $i$ 和 $j$，满足 $a_{ij} + a_{ji} \le 0$。
    * 对于任意两个不同的行 $i$ 和 $j$，满足 $a_{ij} \le -c$，其中 $c$ 为一个正数。

### 2.3 核心概念联系

带有“性质c”的M-矩阵是M-矩阵的一个特殊类型，其定义是在传统M-矩阵定义的基础上增加了非对角元素的约束条件。这种约束条件使得带有“性质c”的M-矩阵具有更强的性质，例如更强的非奇异性、更小的谱半径等。

import numpy as np

def is_M_matrix_with_property_c(A, c):
  """
  判定一个矩阵是否为带有“性质c”的M-矩阵。

  Args:
    A: 矩阵。
    c: “性质c”的约束条件。

  Returns:
    True 如果矩阵为带有“性质c”的M-矩阵，否则为 False。
  """

  # 判定矩阵是否为M-矩阵
  if not np.all(np.diag(A) > 0):
    return False
  if not np.all(A[np.triu_indices(A.shape[0], 1)] <= 0):
    return False
  try:
    A_inv = np.linalg.inv(A)
    if not np.all(A_inv >= 0):
      return False
  except np.linalg.LinAlgError:
    return False

  # 判定矩阵是否满足“性质c”的约束条件
  if c == 'a':
    if not np.all(A[np.triu_indices(A.shape[0], 1)] + A[np.tril_indices(A.shape[0], -1)] <= 0):
      return False
  elif c == 'b':
    if not np.all(A[np.triu_indices(A.shape[0], 1)] <= -c):
      return False
  else:
    return False

  return True

# 例子：

              [-1, 2, -1],
              [0, -1, 2]])
