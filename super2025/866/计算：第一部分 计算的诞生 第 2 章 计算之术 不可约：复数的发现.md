## 2. 核心概念与联系

复数是形如 $a + bi$ 的数，其中 $a$ 和 $b$ 是实数，$i$ 是虚数单位，满足 $i^2 = -1$。复数在数学上具有丰富的性质，与实数之间存在着紧密的联系。

### 2.1 复数的定义

复数可以看作是实数平面上的点，其中实部 $a$ 代表横坐标，虚部 $b$ 代表纵坐标。复数可以通过以下方式表示：

$$
a + bi = (a, b)
$$

### 2.2 复数的运算

复数的运算遵循以下规则：

- 加法：$(a + bi) + (c + di) = (a + c) + (b + d)i$
- 减法：$(a + bi) - (c + di) = (a - c) + (b - d)i$
- 乘法：$(a + bi)(c + di) = (ac - bd) + (ad + bc)i$
- 除法：$\frac{a + bi}{c + di} = \frac{(a + bi)(c - di)}{c^2 + d^2} = \frac{(ac + bd) + (bc - ad)i}{c^2 + d^2}$

### 2.3 复数的几何意义

复数在复平面上可以表示为一个点，其中实部 $a$ 代表横坐标，虚部 $b$ 代表纵坐标。复数的运算可以看作是复平面上点的运算。

## 4. 数学模型和公式 & 详细讲解 & 举例说明
### 4.1 数学模型构建

复数的数学模型可以看作是实数向量的扩展。复数 $a + bi$ 可以看作是一个二维向量，其中实部 $a$ 代表横坐标，虚部 $b$ 代表纵坐标。

### 4.2 公式推导过程

复数的运算是基于实数运算的扩展。以下为复数乘法的推导过程：

假设有两个复数 $a + bi$ 和 $c + di$，它们的乘积为：

$$
(a + bi)(c + di) = (ac - bd) + (ad + bc)i
$$

证明如下：

$$
\begin{aligned}
(a + bi)(c + di) &= ac + adi + bci + bdi^2 \
&= ac + adi + bci - bd \
&= (ac - bd) + (ad + bc)i
\end{aligned}
$$

### 4.3 案例分析与讲解

以下为一个复数乘法案例：

计算 $(2 + 3i)(4 + 5i)$。

解：

$$
(2 + 3i)(4 + 5i) = 8 + 10i + 12i + 15i^2 = 8 + 22i - 15 = -7 + 22i
$$

### 4.4 常见问题解答

**Q1：为什么复数乘法中要使用 $i^2 = -1$？**

A：$i^2 = -1$ 是为了保持复数运算的封闭性。如果 $i^2$ 不等于 $-1$，那么复数运算的结果就可能是实数或虚数，这将导致复数运算不封闭。

### 5.2 源代码详细实现

以下是一个复数乘法实例的代码实现：

```python
# 定义复数类
class ComplexNumber:
    def __init__(self, real, imag):
        self.real = real
        self.imag = imag

    def __add__(self, other):
        return ComplexNumber(self.real + other.real, self.imag + other.imag)

    def __mul__(self, other):
        return ComplexNumber(self.real * other.real - self.imag * other.imag,
                            self.real * other.imag + self.imag * other.real)

    def __str__(self):
        return f"{self.real} + {self.imag}i"

# 创建复数
a = ComplexNumber(2, 3)
b = ComplexNumber(4, 5)

# 计算乘积
result = a * b

# 输出结果
print(result)
```

- `__add__` 方法：实现复数的加法运算。
- `__mul__` 方法：实现复数的乘法运算。
- `__str__` 方法：将复数转换为字符串格式。

运行上述代码，输出结果为：
