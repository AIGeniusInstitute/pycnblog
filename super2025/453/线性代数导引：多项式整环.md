## 2. 核心概念与联系
### 2.1 多项式整环
多项式整环是抽象代数中一个重要的概念，它指的是一个包含多项式的环，其中多项式系数来自一个域或整环。

* **域:** 满足加法、减法、乘法和除法运算的集合。
* **整环:** 满足加法、减法、乘法运算的集合，并且满足交换律、结合律和分配律。

例如，实数域 $\mathbb{R}$ 和整数环 $\mathbb{Z}$ 都是域和整环。

### 2.2 线性代数基础
为了理解线性代数在多项式运算中的应用，我们需要掌握一些基本的线性代数知识，例如：

* **向量空间:**  一个集合加上两个运算（加法和标量乘法）构成向量空间，其中满足一定的公理。
* **线性变换:**  从一个向量空间到另一个向量空间的映射，满足线性性质。
* **矩阵:**  一个数的集合，按照行和列排列，可以表示线性变换。
* **行列式:**  一个数，可以用来判断矩阵是否可逆。
* **逆矩阵:**  一个矩阵，与另一个矩阵相乘得到单位矩阵。

### 2.3 多项式与向量空间
我们可以将多项式看作向量空间中的向量。例如，对于一个域 $F$，所有 $n$ 次多项式组成的集合 $F[x]_n$ 可以构成一个 $n+1$ 维向量空间。

* **向量加法:**  两个多项式的加法对应于系数的加法。
* **标量乘法:**  一个标量乘以一个多项式对应于每个系数都乘以该标量。

## 3. 核心算法原理 & 具体操作步骤
### 3.1  算法原理概述
线性代数在多项式运算中的应用主要体现在以下几个方面：

* **多项式表示:** 将多项式表示为向量，方便进行矩阵运算。
* **多项式加减法:** 利用向量加减法实现多项式加减法。
* **多项式乘法:** 利用矩阵乘法实现多项式乘法。
* **多项式求根:** 利用线性代数的求解线性方程组的方法求解多项式方程的根。

### 3.2  算法步骤详解
#### 3.2.1 多项式表示
将一个 $n$ 次多项式 $p(x) = a_nx^n + a_{n-1}x^{n-1} + ... + a_1x + a_0$ 表示为向量 $[a_n, a_{n-1}, ..., a_1, a_0]$。

#### 3.2.2 多项式加减法
两个多项式的加减法可以通过向量加减法实现。例如，

$p(x) + q(x) = [a_n, a_{n-1}, ..., a_1, a_0] + [b_n, b_{n-1}, ..., b_1, b_0] = [a_n + b_n, a_{n-1} + b_{n-1}, ..., a_1 + b_1, a_0 + b_0]$

#### 3.2.3 多项式乘法
多项式乘法可以通过矩阵乘法实现。

* 构建一个 Vandermonde 矩阵 $V$，其中每一行对应一个不同的 $x$ 值，例如：

```
V = [[x^n, x^(n-1), ..., x, 1],
     [x^(n+1), x^n, ..., x, 1],
     ...,
     [x^(2n), x^(2n-1), ..., x, 1]]
```

* 将多项式系数向量 $p$ 和 $q$ 作为列向量，分别记为 $p_c$ 和 $q_c$。

* 则多项式乘积 $p(x)q(x)$ 的系数向量可以表示为：

```
p(x)q(x) = V * p_c * q_c^T
```

其中 $q_c^T$ 是 $q_c$ 的转置向量。

#### 3.2.4 多项式求根
利用线性代数的求解线性方程组的方法求解多项式方程的根。例如，对于一个 $n$ 次多项式 $p(x)$，我们可以将其表示为：

```
p(x) = a_nx^n + a_{n-1}x^{n-1} + ... + a_1x + a_0 = 0
```

我们可以将这个方程看作一个线性方程组，其中未知数是 $x$ 的值。利用线性代数的求解线性方程组的方法，我们可以找到 $x$ 的根。

### 3.3  算法优缺点
#### 3.3.1 优点
* **效率:** 利用矩阵运算可以提高多项式运算的效率。
* **精度:** 避免了浮点数运算的精度损失。
* **通用性:** 可以应用于各种类型的多项式。

#### 3.3.2 缺点
* **复杂度:** 对于高阶多项式，矩阵运算的复杂度可能会很高。
* **内存占用:** 存储矩阵需要较大的内存空间。


计算结果为 $p(x)q(x) = x^3 + x^2 - x - 1$。

### 5.2  源代码详细实现
```python
import numpy as np

def polynomial_add(p, q):
  """
  计算两个多项式的和。

  Args:
    p: 第一个多项式的系数向量。
    q: 第二个多项式的系数向量。

  Returns:
    两个多项式的和的系数向量。
  """
  return np.add(p, q)

def polynomial_subtract(p, q):
  """
  计算两个多项式的差。

  Args:
    p: 第一个多项式的系数向量。
    q: 第二个多项式的系数向量。

  Returns:
    两个多项式的差的系数向量。
  """
  return np.subtract(p, q)

def polynomial_multiply(p, q):
  """
  计算两个多项式的乘积。

  Args:
    p: 第一个多项式的系数向量。
    q: 第二个多项式的系数向量。

  Returns:
    两个多项式的乘积的系数向量。
  """
  n = len(p)
  m = len(q)
  V = np.zeros((n + m, n + 1))
  for i in range(n + m):
    for j in range(n + 1):
      if j < n:
        V[i, j] = np.power(i, j)
  return np.dot(V, p) * np.transpose(q)

# 示例代码
p = np.array([1, 2, 1])
q = np.array([1, -1])

print("p(x) =", p)
print("q(x) =", q)

p_plus_q = polynomial_add(p, q)
print("p(x) + q(x) =", p_plus_q)

p_minus_q = polynomial_subtract(p, q)
print("p(x) - q(x) =", p_minus_q)

p_times_q = polynomial_multiply(p, q)
print("p(x) * q(x) =", p_times_q)
```

* 然后，我们定义了两个多项式 $p$ 和 $q$ 的系数向量。
* 最后，我们调用三个函数计算多项式的加法、减法和乘法，并打印结果。
