### 4.2 举例说明

**例1：概率与统计**

题目：给定一个包含n个元素的数组，每个元素的范围是[0, 1]，请实现一个函数，计算数组中元素大于阈值的概率。

解题思路：

1. **明确问题**：计算数组中元素大于阈值的概率。
2. **分析数据结构**：数组。
3. **设计算法**：利用概率统计方法，计算大于阈值的元素个数，再除以总元素个数。
4. **编写代码**：实现算法并编写代码。

代码实现：

```python
def probability_of_threshold(arr, threshold):
    count = 0
    for num in arr:
        if num > threshold:
            count += 1
    return count / len(arr)
```

**例2：线性代数**

题目：给定一个m×n的矩阵，请实现一个函数，计算矩阵的行列式。

解题思路：

1. **明确问题**：计算矩阵的行列式。
2. **分析数据结构**：矩阵。
3. **设计算法**：利用递归方法，将矩阵分解为子矩阵，计算子矩阵的行列式。
4. **编写代码**：实现算法并编写代码。

代码实现：

```python
def determinant(matrix):
    if len(matrix) == 1:
        return matrix[0][0]
    if len(matrix) == 2:
        return matrix[0][0] * matrix[1][1] - matrix[0][1] * matrix[1][0]
    det = 0
    for col in range(len(matrix[0])):
        sub_matrix = [[row[i] for i in range(len(matrix[0]))] for row in matrix[1:]]
        det += ((-1) ** col) * matrix[0][col] * determinant(sub_matrix)
    return det
```

### 5.2 源代码详细实现

以下是一个简单的算法题代码实例，题目为“求两个数的最大公约数”。

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

num1 = 24
num2 = 36
print(gcd(num1, num2))
```

### 5.3 代码解读与分析

1. **函数定义**：定义一个名为`gcd`的函数，接收两个整数参数`a`和`b`。
2. **循环语句**：使用`while`循环，当`b`不为0时，执行循环体。
3. **变量交换**：将`b`的值赋给`a`，将`a`除以`b`的余数赋给`b`。
4. **返回值**：当`b`为0时，跳出循环，返回`a`的值，即为最大公约数。

### 5.4 运行结果展示

运行上述代码，输出结果为`12`，即24和36的最大公约数为12。
