@RestController
@RequestMapping("/courses")
public class CourseController {

    @Autowired
    private CourseService courseService;

    @GetMapping
    public ResponseEntity<List<Course>> getAllCourses() {
        return ResponseEntity.ok(courseService.findAll());
    }

    @GetMapping("/{id}")
    public ResponseEntity<Course> getCourseById(@PathVariable Long id) {
        Optional<Course> course = courseService.findById(id);
        if (course.isPresent()) {
            return ResponseEntity.ok(course.get());
        }
        return ResponseEntity.notFound().build();
    }

    @GetMapping("/search")
    public ResponseEntity<List<Course>> searchCourses(@RequestParam String query) {
        return ResponseEntity.ok(courseService.searchByTitle(query));
    }
}

### 附录：常见问题与解答

#### 问题 1：如何选择创业方向？

解答：选择创业方向需要考虑以下几个因素：

1. **个人兴趣**：选择自己感兴趣且愿意长期投入的领域。
2. **市场需求**：分析目标市场的需求，选择具有市场潜力的方向。
3. **竞争优势**：了解自己在该领域的竞争优势，如技术、经验等。
4. **资源条件**：评估自身的资源，如资金、团队、人脉等。

#### 问题 2：创业过程中如何管理资金？

解答：创业过程中管理资金需要注意以下几点：

1. **预算规划**：制定详细的预算计划，合理分配资金。
2. **成本控制**：严格控制各项成本，避免不必要的浪费。
3. **资金流转**：确保资金的合理流转，避免资金链断裂。
4. **融资策略**：根据实际情况，选择合适的融资渠道和策略。

#### 问题 3：创业团队如何建设？

解答：建设创业团队需要遵循以下几个原则：

1. **明确目标**：确保团队成员对项目目标和愿景有共同的认识。
2. **优势互补**：选择具备不同技能和经验的成员，实现优势互补。
3. **信任与沟通**：建立团队成员之间的信任，保持良好的沟通。
4. **激励机制**：设计合理的激励机制，激发团队成员的积极性和创造力。

### Appendix: Frequently Asked Questions and Answers
#### Question 1: How to choose the direction for entrepreneurship?

Answer: Choosing the direction for entrepreneurship requires considering the following factors:

1. **Personal Interest**: Choose a field that you are interested in and willing to commit to long-term.
2. **Market Demand**: Analyze the demand in the target market and choose a direction with market potential.
3. **Competitive Advantage**: Understand your competitive advantages in the field, such as technology or experience.
4. **Resource Conditions**: Assess your own resources, such as funds, team, and connections.

#### Question 2: How to manage funds during the entrepreneurial process?

Answer: To manage funds during the entrepreneurial process, you should pay attention to the following points:

1. **Budget Planning**: Develop a detailed budget plan and allocate funds reasonably.
2. **Cost Control**: Strictly control costs to avoid unnecessary waste.
3. **Fund Flow**: Ensure the reasonable flow of funds to avoid a broken cash flow.
4. **Funding Strategies**: Choose appropriate funding channels and strategies based on the actual situation.

#### Question 3: How to build an entrepreneurial team?

Answer: Building an entrepreneurial team should follow these principles:

1. **Clear Goals**: Ensure that team members have a common understanding of the project goals and vision.
2. **Complementary Strengths**: Choose team members with different skills and experiences to achieve complementary strengths.
3. **Trust and Communication**: Build trust among team members and maintain good communication.
4. **Incentive Mechanisms**: Design a reasonable incentive system to stimulate the enthusiasm and creativity of team members.
