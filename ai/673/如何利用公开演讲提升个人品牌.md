### 5.2 源代码详细实现

以下是公开演讲管理系统的源代码实现，包括演讲信息录入、效果评估和品牌提升跟踪等功能。

```python
import pandas as pd

# 演讲信息录入
def input_speech_info():
    speech_info = {
        'title': input('请输入演讲标题：'),
        'date': input('请输入演讲日期：'),
        'score': input('请输入演讲效果得分：'),
        'contacts': input('请输入结识的新联系人数量：')
    }
    return speech_info

# 演讲效果评估
def evaluate_speech(speech_info):
    speech_score = speech_info['score']
    speech_effects = {
        'language_expression': speech_score * 0.4,
        'speech_tech': speech_score * 0.5,
        'emotional_resonance': speech_score * 0.1
    }
    return speech_effects

# 个人品牌提升跟踪
def track_brand(speech_effects, contacts):
    brand_speed = speech_effects['language_expression'] * speech_effects['speech_tech'] * speech_effects['emotional_resonance'] * contacts
    return brand_speed

# 主程序
def main():
    speeches = []
    while True:
        info = input_speech_info()
        if info['title'] == '结束':
            break
        speeches.append(info)
    
    for speech in speeches:
        speech_effects = evaluate_speech(speech)
        brand_speed = track_brand(speech_effects, speech['contacts'])
        print('演讲效果评估：', speech_effects)
        print('个人品牌提升速度：', brand_speed)

    df = pd.DataFrame(speeches)
    print('演讲记录：')
    print(df)

if __name__ == '__main__':
    main()
```

2. 演讲效果评估：根据效果得分计算语言表达、演讲技巧和情感共鸣等效果得分。
