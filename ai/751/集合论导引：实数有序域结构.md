### 1.1 集合论的基本概念

#### 集合（Set）

集合是数学中一个基本的概念，它是由确定性的元素组成的整体。集合通常用大写字母表示，例如A、B、C等。集合中的元素可以是任何对象，包括数字、符号、点、线、面等。集合的元素可以是有限的，也可以是无限的。

#### 子集（Subset）

如果一个集合A的所有元素都属于另一个集合B，则称A是B的子集，记作A⊆B。例如，集合{1, 2, 3}是集合{1, 2, 3, 4, 5}的子集。

#### 集合运算（Set Operations）

集合运算包括并集（Union）、交集（Intersection）和补集（Complement）等。

- 并集（Union）：A和B的并集是由属于A或属于B的元素组成的集合，记作A∪B。
- 交集（Intersection）：A和B的交集是由同时属于A和B的元素组成的集合，记作A∩B。
- 补集（Complement）：A的补集是由不属于A的元素组成的集合，记作A'。

#### 映射（Mapping）

映射是一种将一个集合中的元素映射到另一个集合中的函数。映射通常用小写字母表示，例如f、g、h等。映射可以看作是一种特殊的函数，它将集合中的每个元素与另一个集合中的唯一元素相对应。

### 3.1 集合运算算法

集合运算算法包括并集、交集和补集等操作。以下是这些算法的具体操作步骤：

#### 3.1.1 并集算法

1. 创建一个空集合A。
2. 对于集合B中的每个元素，如果该元素不在集合A中，则将其添加到集合A。
3. 返回集合A。

#### 3.1.2 交集算法

1. 创建一个空集合C。
2. 对于集合A和集合B中的每个元素，如果该元素同时存在于集合A和集合B中，则将其添加到集合C。
3. 返回集合C。

#### 3.1.3 补集算法

1. 创建一个空集合D。
2. 对于集合A中的每个元素，如果该元素不在集合B中，则将其添加到集合D。
3. 返回集合D。

### 3.2 实数有序域排序算法

实数有序域排序算法用于将实数集合中的元素按照大小进行排序。以下是两种常见的排序算法：快速排序和归并排序。

#### 3.2.1 快速排序算法

1. 选择集合中的一个元素作为基准（pivot）。
2. 将集合中的其他元素分为两部分，一部分小于基准，另一部分大于基准。
3. 对小于基准的子集和大于基准的子集递归执行快速排序。
4. 将排序好的子集合并，得到排序好的集合。

#### 3.2.2 归并排序算法

1. 将集合分为两个大小相等的子集。
2. 对每个子集递归执行归并排序。
3. 将两个排序好的子集合并，得到排序好的集合。

### 3.3 集合运算与排序算法的关联

def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

# 5.2.3 实数有序域的度量

def distance(x, y):
