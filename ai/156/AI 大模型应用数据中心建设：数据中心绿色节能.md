使用最小二乘法估计模型参数$\beta_0, \beta_1, \beta_2, \beta_3, \beta_4$，公式推导过程如下：

$$\hat{\beta} = (X^TX)^{-1}X^TE$$

其中，$X$表示特征矩阵，$E$表示能耗向量，$\hat{\beta}$表示模型参数估计值。

* Matplotlib：数据可视化库

### 5.2 源代码详细实现

以下是使用线性回归模型构建能耗预测模型的源代码实现：

```python
import numpy as np
import pandas as pd
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error

# 加载数据
data = pd.read_csv('data.csv')

# 特征工程
X = data[['temperature', 'humidity', 'utilization', 'parameters']]
y = data['energy']

# 构建模型
model = LinearRegression()

# 训练模型
model.fit(X, y)

# 评估模型
y_pred = model.predict(X)
mse = mean_squared_error(y, y_pred)
print(f'Mean Squared Error: {mse}')

# 预测能耗
X_test = np.array([[29, 60, 0.85, 10**8]])
y_test = model.predict(X_test)
print(f'Predicted Energy: {y_test[0]}')
```
