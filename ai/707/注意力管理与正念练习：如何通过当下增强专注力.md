## 3. 核心算法原理 & 具体操作步骤

### 正念呼吸练习

正念呼吸练习是一种简单而有效的正念练习方法，可以帮助我们增强专注力。以下是一个具体的操作步骤：

1. 找一个安静的地方，坐直或舒适地站立。
2. 将注意力集中在你的呼吸上。感受空气通过鼻孔进入你的身体，然后从嘴巴或鼻子呼出。
3. 深呼吸几次，让呼吸变得深而平稳。
4. 当你的注意力开始分散时，轻轻地将其引导回呼吸上。
5. 继续这个过程，每次呼吸都是一个宝贵的机会，让你重新集中注意力。

### 正念行走练习

正念行走练习可以帮助我们在活动中保持专注。以下是具体的操作步骤：

1. 在一个相对安静的环境中开始行走。
2. 注意你的步伐，感受脚与地面接触的感觉。
3. 注意你的身体，感受身体的重心和运动。
4. 将注意力集中在呼吸上，感受空气的流动。
5. 当你的注意力开始分散时，轻轻地将其引导回到行走和呼吸上。

### 正念饮食练习

正念饮食练习可以帮助我们提高对食物的觉知，从而更好地控制食欲和饮食。以下是具体的操作步骤：

1. 在进餐时，放慢速度，专注于食物的味道和质感。
2. 注意你的咀嚼和吞咽，感受食物在嘴里的味道。
3. 感受身体对食物的需求和满足感。
4. 当你感到饱腹时，停止进食，感受身体对食物的感激。

### 5.2 源代码详细实现

以下是一个简单的 Python 脚本，用于记录和可视化正念练习的时间安排：

```python
import numpy as np
import matplotlib.pyplot as plt

def plot_time_schedule(hours_per_day, hours_for_practice):
    time_available = hours_per_day - hours_for_practice
    time_for_practice = [hours_for_practice] * 7
    time_for_other_activities = [time_available] * 7
    
    plt.bar(range(7), time_for_practice, width=0.4, label='正念练习')
    plt.bar(range(7), time_for_other_activities, width=0.4, label='其他活动')
    plt.xticks(range(7), ['周一', '周二', '周三', '周四', '周五', '周六', '周日'])
    plt.xlabel('天数')
    plt.ylabel('时间（小时）')
    plt.title('每天的时间安排')
    plt.legend()
    plt.show()

# 设置参数
hours_per_day = 24
hours_for_practice = 1

# 绘制时间安排图
plot_time_schedule(hours_per_day, hours_for_practice)
```

### 5.3 代码解读与分析

上述代码定义了一个名为 `plot_time_schedule` 的函数，用于计算和绘制每天的时间安排。函数接受两个参数：`hours_per_day`（每天的总小时数）和 `hours_for_practice`（用于正念练习的小时数）。函数首先计算每天剩余的时间 `time_available`，然后创建两个列表 `time_for_practice` 和 `time_for_other_activities`，分别表示每天用于正念练习和其他活动的时间。

在绘制部分，我们使用 `plt.bar` 函数创建一个条形图，其中 `range(7)` 生成一周的七天，`width=0.4` 设置条形图的宽度，`label` 设置图例的标签。最后，我们使用 `plt.xticks` 设置 x 轴标签，`plt.xlabel` 和 `plt.ylabel` 设置 x 轴和 y 轴的标签，`plt.title` 设置图表标题，`plt.legend` 添加图例，`plt.show` 显示图表。

### 5.4 运行结果展示

运行上述代码后，将显示一个条形图，展示每天用于正念练习和其他活动的时间分布。这个图表可以帮助我们直观地了解每天的时间安排，从而更好地管理我们的注意力。
