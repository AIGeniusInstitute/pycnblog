## 2. 核心概念与联系
### 2.1  M-矩阵
M-矩阵是一种特殊的非负矩阵，其定义如下：

* **定义:** 设A为n×n的实矩阵，如果A满足以下条件：
    * A的元素都是非负数(A≥0)。
    * A的逆矩阵存在且是正的(A⁻¹>0)。

则称A为M-矩阵。

### 2.2  性质c
“性质c”是指M-矩阵A满足以下条件：

* **定义:** 设A为M-矩阵，如果存在一个非负向量x，使得Ax=x，则称A具有性质c。

### 2.3  带有“性质c”的M-矩阵
带有“性质c”的M-矩阵是指满足以下条件的M-矩阵：

* **定义:** 设A为M-矩阵，如果A具有性质c，则称A为带有“性质c”的M-矩阵。
