## 2. 核心概念与联系

LangChain编程的核心概念包括：

- **事件驱动**：程序通过事件来响应外部输入，而不是通过轮询或阻塞调用。
- **异步编程**：程序在等待外部操作完成时不会阻塞，从而提高效率。
- **架构模式**：如观察者模式、发布-订阅模式等，用于实现消息的发送和接收。
