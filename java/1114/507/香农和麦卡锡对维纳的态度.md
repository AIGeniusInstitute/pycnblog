以下是一个使用 NumPy 库实现香农的信息熵的 Python 代码示例：

```python
import numpy as np

def entropy(p):
    """计算信息熵"""
    return -np.sum(p * np.log2(p))

# 示例：计算一个概率分布的信息熵
probs = np.array([0.9, 0.1])
print("Information entropy:", entropy(probs))
```
