### 4.2 公式推导过程

#### 4.2.1 甘特图

甘特图是一种以时间为基础的项目管理工具，用于表示项目活动的开始和结束时间。甘特图的公式推导过程如下：

$$
\begin{aligned}
&Gantt\_Chart(A_{ij}) = \{ \text{活动} A_i \text{，时间} [t_{i1}, t_{if}] \} \
\text{其中：} \
&A_i \text{为活动} \
&t_{i1} \text{为活动} A_i \text{的开始时间} \
&t_{if} \text{为活动} A_i \text{的结束时间}
\end{aligned}
$$

#### 4.2.2 关键路径法(CPM)

关键路径法是一种项目进度管理方法，用于确定项目完成所需的最短时间。CPM的公式推导过程如下：

$$
\begin{aligned}
&CPM(A_{ij}) = t_{ij} + t_{j+1} \
\text{其中：} \
&A_{ij} \text{为活动} \
&t_{ij} \text{为活动} A_{ij} \text{的持续时间} \
&t_{j+1} \text{为活动} A_{ij} \text{的后续活动持续时间}
\end{aligned}
$$

### 4.3 案例分析与讲解

#### 4.3.1 项目进度模型案例分析

以下是一个甘特图案例：

| 活动 | 开始时间 | 结束时间 |
| :--: | :----: | :----: |
| A    |  1     |  3     |
| B    |  2     |  4     |
| C    |  4     |  7     |
| D    |  5     |  8     |
| E    |  6     |  9     |

根据甘特图，可以计算出关键路径：

1. A-B-C-E：总时长为9天
2. A-B-D-E：总时长为9天

因此，关键路径为A-B-C-E和A-B-D-E。

#### 4.3.2 项目成本模型案例分析

以下是一个关键路径法(CPM)案例：

| 活动 | 持续时间 | 直接成本 |
| :--: | :----: | :----: |
| A    |   5    |  100   |
| B    |   3    |  200   |
| C    |   2    |  300   |
| D    |   4    |  400   |
| E    |   1    |  500   |

根据CPM，可以计算出关键路径的总成本：

1. A-B-C-E：总成本为100 + 200 + 300 + 500 = 1100
2. A-B-D-E：总成本为100 + 200 + 400 + 500 = 1200

因此，关键路径的总成本为1100。

### 4.4 常见问题解答

**Q1：甘特图和PERT图的区别是什么？**

A：甘特图以时间为横坐标，以活动为纵坐标，直观地展示项目进度。PERT图则以活动为横坐标，以时间为纵坐标，并用三个时间值表示活动的最早开始时间、最晚开始时间和最可能时间。

**Q2：关键路径法(CPM)如何确定关键路径？**

A：关键路径是项目完成所需的最短时间路径。确定关键路径的方法包括：1）计算每个活动的最早开始时间、最晚开始时间、最早结束时间和最晚结束时间；2）计算每个活动的总浮动时间；3）确定总浮动时间为零的活动，这些活动构成关键路径。

### 5.2 源代码详细实现

以下是一个简单的项目进度管理代码实例：

```python
import numpy as np
import matplotlib.pyplot as plt

def gantt_chart(activities, start_times, end_times):
    fig, ax = plt.subplots(figsize=(10, 8))
    for i in range(len(activities)):
        ax.plot([start_times[i], end_times[i]], [i, i], color='black', linewidth=2)
    ax.set_xlim([min(start_times), max(end_times)])
    ax.set_ylim([0, len(activities)])
    ax.set_xticks(range(min(start_times), max(end_times) + 1, 1))
    ax.set_xticklabels(start_times, rotation=90)
    ax.set_yticks(range(len(activities)))
    ax.set_yticklabels(activities)
    plt.show()

# 项目活动列表
activities = ['活动A', '活动B', '活动C', '活动D', '活动E']
# 活动开始时间
start_times = [1, 2, 4, 5, 6]
# 活动结束时间
end_times = [3, 4, 7, 8, 9]

gantt_chart(activities, start_times, end_times)
```

### 5.3 代码解读与分析

该代码首先导入了numpy和matplotlib库，用于数学计算和绘图。`gantt_chart`函数接受活动列表、活动开始时间和结束时间作为输入，绘制甘特图。代码中，使用matplotlib的`subplot`函数创建一个图形，并使用`plot`函数绘制活动条形。最后，使用`show`函数显示甘特图。
