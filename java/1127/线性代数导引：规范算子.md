## 2. 核心概念与联系

规范算子是线性代数中的一个重要概念。在一个内积空间中，如果一个线性算子$A$满足$A^*A = AA^* = I$，那么我们称$A$为规范算子，其中$A^*$是$A$的共轭转置，$I$是单位矩阵。规范算子的一个重要性质是它保持内积不变，即对任意的向量$x$，$y$，我们有$(Ax, Ay) = (x, y)$。这个性质使得规范算子在许多应用中都非常有用。

规范算子的优点主要有两个：一是它可以将复杂的问题简化，使得问题更容易解决；二是它保持内积不变，这使得我们可以在不改变原问题性质的情况下，对问题进行简化。
