## 2. 核心概念与联系

### 2.1 格与复环面的定义

**格**：格是一种代数结构，它是一种部分有序集，满足以下条件：
1. 有两个二元运算：加法和乘法。
2. 加法和乘法满足结合律。
3. 存在零元和单位元。
4. 加法满足交换律。
5. 乘法满足交换律。
6. 对于每个元素 $a$，存在逆元 $-a$。

**复环面**：复环面是格的一个特殊例子，它满足以下条件：
1. 它是一个格。
2. 它的乘法运算满足分配律。

### 2.2 格与复环面的联系

复环面是一种特殊的格，因此它也满足格的所有性质。
