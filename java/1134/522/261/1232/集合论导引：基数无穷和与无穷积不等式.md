## 2. 核心概念与联系

### 2.1 核心概念概述

在集合论中，基数是表示集合中元素数量的概念。如果集合$A$和集合$B$中元素数量相等，那么称集合$A$和集合$B$有相同的基数。集合的基数可以通过集合的元素个数、元素的序数等方式来表示。

无穷集合是指具有无限多个元素的集合。常见的无穷集合有自然数集、整数集、有理数集、实数集等。无穷集合的基数是无限的，无法用有限的数字来表示。

无穷积是指两个或多个无穷集合的笛卡尔积的基数。例如，如果$A$和$B$是两个无穷集合，则$A \times B$的基数等于$|A| \cdot |B|$，其中$|A|$和$|B|$分别表示集合$A$和$B$的基数。
