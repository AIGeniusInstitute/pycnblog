### 5.2 源代码详细实现

以下是一些代码示例：

```python
# 计算自然数的和
def sum_natural_numbers(n):
  """
  计算从1到n的自然数的和。

  Args:
    n: 自然数的上限。

  Returns:
    从1到n的自然数的和。
  """
  sum = 0
  for i in range(1, n + 1):
    sum += i
  return sum

# 测试函数
n = 100
sum = sum_natural_numbers(n)
print(f"从1到{n}的自然数的和为：{sum}")
```

这段代码实现了计算从1到n的自然数的和的功能。代码使用循环来遍历从1到n的每个自然数，并将它们累加到sum变量中。最后，代码返回sum变量的值。
