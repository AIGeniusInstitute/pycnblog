## 2. 核心概念与联系

### 2.1 通用人工智能

麦卡锡是“通用人工智能”概念的提出者，他认为，人工智能的目标应该是创造出能够像人类一样思考和学习的机器。这种机器应该能够解决各种问题，并能够适应不同的环境。麦卡锡的“通用人工智能”概念，为人工智能研究指明了方向，也为人工智能发展提供了终极目标。

### 2.2 框架理论

明斯基提出的“框架理论”，是人工智能领域中重要的知识表示和推理理论之一。框架理论认为，知识可以被组织成一系列的框架，每个框架代表一个概念或一个物体。框架包含一系列的槽，每个槽代表一个属性或一个关系。通过对框架的填充和操作，可以进行知识推理和问题求解。

### 2.3 合作与影响

麦卡锡和明斯基在人工智能领域有着密切的合作关系，他们共同创办了麻省理工学院人工智能实验室（MIT AI Lab），并共同领导了人工智能领域早期的一些重要研究项目。他们的合作，促进了人工智能领域的发展，也为人工智能研究奠定了重要的理论基础。

```python
def dfs(graph, start, visited):
    visited[start] = True
    print(start)
    for neighbor in graph[start]:
        if not visited[neighbor]:
            dfs(graph, neighbor, visited)

graph = {
    'A': ['B', 'C'],
    'B': ['D', 'E'],
    'C': ['F'],
    'D': [],
    'E': [],
    'F': []
}

visited = [False] * len(graph)
dfs(graph, 'A', visited)
```
