## 2. 核心概念与联系

### 2.1 勒贝格可测集

勒贝格可测集是勒贝格测度理论中的基本概念。一个集合被称为勒贝格可测集，如果它满足以下条件：

* **可测性：** 该集合的测度可以被定义，即可以计算它的“大小”。
* **可加性：** 对于该集合的任何两个不相交子集，它们的测度之和等于这两个子集的并集的测度。
* **完备性：** 该集合的任何子集的测度都存在，即使该子集本身不可测。

### 2.2 勒贝格测度

勒贝格测度是定义在勒贝格可测集上的一个函数，它将每个可测集映射到一个非负实数，称为该集合的测度。勒贝格测度满足以下性质：

* **非负性：** 任何可测集的测度都是非负的。
* **可加性：** 对于任何两个不相交的可测集，它们的测度的和等于它们的并集的测度。
* **单调性：** 如果一个可测集包含另一个可测集，那么前者的测度大于或等于后者的测度。

### 2.3 勒贝格可测性与其他数学概念的联系
