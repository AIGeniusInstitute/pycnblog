## 2. 核心概念与联系

### 2.1 微分几何

微分几何主要研究在光滑流形上的几何性质。流形是一个局部与欧几里得空间同胚的几何空间。以下是微分几何中的几个关键概念：

- **度量**: 描述了流形上的距离和角度。
- **切空间**: 在流形上的每个点都有一个切空间，它是所有在该点处切线的集合。
- **向量场**: 在流形上定义的场，它在每个点都有一个向量。
- **测地线**: 在流形上两点之间距离最短的曲线。

### 2.2 黎曼几何

黎曼几何是微分几何的一个分支，它研究带有一致度量张量的流形。黎曼几何的几个核心概念包括：

- **黎曼度规**: 描述了流形上的度量。
- **黎曼曲率**: 描述了流形的弯曲程度。
- **测地线方程**: 描述了在黎曼流形上测地线的运动。

### 2.3 广义相对论与微分几何

广义相对论将重力视为时空的弯曲，而微分几何则为这种弯曲提供了一个数学框架。在广义相对论中，时空被视为一个四维的黎曼流形，其度规由爱因斯坦场方程确定。

```python
import numpy as np

# 定义角动量守恒函数
def angular_momentum_conservation(I, omega, tau, t):
    return I * omega + tau * t

# 初始参数
```
