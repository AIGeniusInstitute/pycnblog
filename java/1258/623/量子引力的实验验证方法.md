## 2. 核心概念与联系

### 2.1 量子力学

量子力学是研究微观粒子的运动规律的物理学分支，其核心概念包括：

- **波粒二象性**：微观粒子既具有波动性，又具有粒子性。
- **量子态**：微观粒子的状态可以用波函数来描述，波函数的平方给出粒子出现在某个位置的概率。
- **叠加原理**：一个微观粒子可以同时处于多个量子态的叠加状态。

### 2.2 广义相对论

广义相对论是研究引力场和物质之间相互作用的物理学理论，其核心概念包括：

- **时空**：时空是物质和引力场共同作用的结果，具有弯曲的性质。
- **引力波**：引力波是时空的波动，是由物质加速运动产生的。
- **黑洞**：黑洞是引力场极强的天体，其事件视界内的物质和辐射无法逃逸。

### 2.3 量子引力

量子引力是试图将量子力学和广义相对论统一起来的理论，其核心概念包括：

- **量子场论**：量子场论是量子力学和广义相对论的桥梁，用于描述物质和场在量子尺度下的相互作用。
- **量子时空**：量子时空是量子场论在引力场下的时空，具有量子效应。
- **量子引力波**：量子引力波是量子时空的波动，由物质和辐射产生。
