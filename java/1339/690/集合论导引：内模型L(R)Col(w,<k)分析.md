- **拓扑空间**：一个拓扑空间是一个集合，它配备了一个拓扑，即一组开集的集合，满足一定的公理。
- **代数结构**：一个代数结构是一个集合，它配备了一个或多个运算，满足一定的公理。
- **同态**：一个同态是一个映射，它保持代数结构的运算。
