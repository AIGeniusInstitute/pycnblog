## 2. 核心概念与联系

### 2.1 流形

流形是一个局部欧几里得空间，即它在每个点都有一个局部邻域与某个欧几里得空间同胚。流形可以是多维的，可以是弯曲的，也可以是复杂的几何形状。流形拓扑学主要研究流形的拓扑性质，如连通性、维数、边界、同伦等。

### 2.2 同调理论

同调理论是拓扑学中的一个重要分支，它研究的是流形的同调类。同调类是流形上的一种不变量，可以用来刻画流形的拓扑结构。同调理论的核心概念包括同调群、边界映射、上同调等。

### 2.3 Bott周期定理

Bott周期定理是流形拓扑学中的一个重要定理，它建立了特征类与同调类之间的关系。该定理指出，对于一个光滑流形$M$，其特征类$\sigma_k(M)$的整数部分等于$M$的$k$维同调群的秩。
