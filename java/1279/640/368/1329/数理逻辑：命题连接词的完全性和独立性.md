## 2. 核心概念与联系

### 2.1 数理逻辑

数理逻辑是一种形式化的数学分支，用于研究推理的有效性和证明的合理性。它将自然语言的推理过程转化为形式化的符号系统，从而避免了自然语言的不确定性和模糊性。

### 2.2 命题连接词

命题连接词是数理逻辑中的基本元素，用于连接简单命题形成复合命题。常见的命题连接词包括：

- **合取（$\land$）**：表示“且”。
- **析取（$\lor$）**：表示“或”。
- **否定（$\neg$）**：表示“非”。
- **条件（$\rightarrow$）**：表示“如果...那么...”。
- **双条件（$\leftrightarrow$）**：表示“当且仅当”。
