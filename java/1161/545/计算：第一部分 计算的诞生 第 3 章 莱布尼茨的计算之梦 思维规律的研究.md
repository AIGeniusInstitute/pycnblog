### 5.2 源代码详细实现

下面是我们的程序的源代码：

```python
def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, n):
        if n % i == 0:
            return False
    return True

n = 13
print(is_prime(n))
```

这个程序首先定义了一个函数`is_prime`，这个函数接受一个整数`n`作为参数，返回这个数是否是素数。然后，程序调用这个函数，判断13是否是素数，并打印结果。

### 5.3 代码解读与分析

这个程序的核心是`is_prime`函数。这个函数首先检查`n`是否小于或等于1，如果是，那么它返回`False`，因为1和所有负数都不是素数。然后，函数用一个循环来检查`n`是否有除了1和它自身之外的因数，如果有，那么它返回`False`，否则，它返回`True`。

### 5.4 运行结果展示
