**距离代价函数：**

距离代价函数计算从起点到每个可能节点的距离，公式如下：

$$
C_d(i, j) = d(i, j)
$$

其中，$C_d(i, j)$表示从节点$i$到节点$j$的距离代价，$d(i, j)$表示从节点$i$到节点$j$的实际距离。

**时间代价函数：**

时间代价函数计算从起点到每个可能节点的耗时，公式如下：

$$
C_t(i, j) = t(i, j)
$$

其中，$C_t(i, j)$表示从节点$i$到节点$j$的时间代价，$t(i, j)$表示从节点$i$到节点$j$的实际耗时。

**总代价函数：**

总代价函数是距离代价函数和时间代价函数的组合，用于评估每个可能路径的总体优劣，公式如下：

$$
C(i, j) = C_d(i, j) + C_t(i, j)
$$

其中，$C(i, j)$表示从节点$i$到节点$j$的总代价。

### 4.2 A*算法

A*算法是一种基于启发式的路径规划算法，通过评估路径的代价函数找到最优路径。其核心思想是利用启发函数$h(n)$估计从节点$n$到终点$g(n)$的实际代价，公式如下：

$$
f(n) = g(n) + h(n)
$$

其中，$f(n)$表示从起点到节点$n$的评估函数，$g(n)$表示从起点到节点$n$的实际代价，$h(n)$表示从节点$n$到终点的启发函数。

### 4.1 Cost Function in Path Planning

In path planning, the cost function is used to evaluate the quality of each possible path. Common cost functions include distance cost function and time cost function.

**Distance Cost Function:**

The distance cost function calculates the distance from the starting point to each possible node, with the formula:

$$
C_d(i, j) = d(i, j)
$$

Where $C_d(i, j)$ represents the distance cost from node $i$ to node $j$, and $d(i, j)$ represents the actual distance from node $i$ to node $j$.

**Time Cost Function:**

The time cost function calculates the time taken from the starting point to each possible node, with the formula:

$$
C_t(i, j) = t(i, j)
$$

Where $C_t(i, j)$ represents the time cost from node $i$ to node $j$, and $t(i, j)$ represents the actual time taken from node $i$ to node $j$.

**Total Cost Function:**

The total cost function is a combination of the distance cost function and the time cost function, used to evaluate the overall quality of each possible path, with the formula:

$$
C(i, j) = C_d(i, j) + C_t(i, j)
$$

Where $C(i, j)$ represents the total cost from node $i$ to node $j$.

### 4.2 A* Algorithm

A* algorithm is a heuristic-based path planning algorithm that finds the optimal path by evaluating the cost function of each possible path. Its core idea is to use the heuristic function $h(n)$ to estimate the actual cost from node $n$ to the goal $g(n)$, with the formula:

$$
f(n) = g(n) + h(n)
$$

Where $f(n)$ represents the evaluation function from the starting point to node $n$, $g(n)$ represents the actual cost from the starting point to node $n$, and $h(n)$ represents the heuristic function from node $n$ to the goal $g(n)$.
