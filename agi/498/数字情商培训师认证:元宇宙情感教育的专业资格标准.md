### 5.2 源代码详细实现

```python
import nltk
from spacy.lang.en import English
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB

# 数据准备
nltk.download('movie_reviews')
nltk.download('stopwords')

def preprocess_text(text):
    # 分词、去除停用词
    doc = English().process(text)
    return ' '.join([token.lemma_ for token in doc if not token.is_stop])

# 训练数据
positive_reviews = [preprocess_text(review) for review in nltk.corpus.movie_reviews.fileids('pos*')]
negative_reviews = [preprocess_text(review) for review in nltk.corpus.movie_reviews.fileids('neg*')]

# 标签
positive_labels = ['positive'] * len(positive_reviews)
negative_labels = ['negative'] * len(negative_reviews)
labels = positive_labels + negative_labels

# 特征提取
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(positive_reviews + negative_reviews)

# 模型训练
model = MultinomialNB()
model.fit(X, labels)

# 情感识别
def recognize_emotion(text):
    processed_text = preprocess_text(text)
    vector = vectorizer.transform([processed_text])
    emotion = model.predict(vector)[0]
    return emotion

# 测试
print(recognize_emotion("I am so happy!"))  # 输出：positive
print(recognize_emotion("I am so sad..."))  # 输出：negative
```

### 5.3 代码解读与分析

- **数据预处理**：使用NLTK和spaCy对文本进行分词和去除停用词处理，以提高模型准确性。
- **特征提取**：使用CountVectorizer将文本转换为词频矩阵，作为模型的输入。
- **模型训练**：使用朴素贝叶斯分类器对文本进行分类。
- **情感识别**：输入用户发言，提取情感向量，利用模型进行情感分类。

### 5.4 运行结果展示

- 正面情感：`I am so happy!` 输出为positive。
- 负面情感：`I am so sad...` 输出为negative。
