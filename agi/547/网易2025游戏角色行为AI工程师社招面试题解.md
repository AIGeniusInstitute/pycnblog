## 4. 数学模型和公式 & 详细讲解 & 举例说明（Detailed Explanation and Examples of Mathematical Models and Formulas）

#### 4.1 决策树数学模型
决策树的核心在于通过数学模型来选择特征和划分节点。以下是决策树的主要数学模型：

- **信息增益（Information Gain）**：
  $$ IG(D, A) = H(D) - \sum_{v\in A} \frac{|D_v|}{|D|} H(D_v) $$
  其中，$H(D)$ 表示数据集 $D$ 的熵，$A$ 表示特征集合，$D_v$ 表示特征 $v$ 的值。

- **基尼不纯度（Gini Impurity）**：
  $$ GI(D, A) = 1 - \sum_{v\in A} \frac{|D_v|}{|D|}^2 $$
  其中，$A$ 表示特征集合，$D_v$ 表示特征 $v$ 的值。

#### 4.2 状态机数学模型
状态机的数学模型主要涉及状态转移概率和状态价值函数：

- **状态转移概率（State Transition Probability）**：
  $$ P(s_i \rightarrow s_j) = \frac{\text{状态 } s_i \text{ 到状态 } s_j \text{ 的转移次数}}{\text{状态 } s_i \text{ 的总转移次数}} $$

- **状态价值函数（State Value Function）**：
  $$ V^*(s_i) = \max_{a \in A} \sum_{s_j \in S} P(s_j|s_i, a) \cdot R(s_i, a, s_j) $$
  其中，$V^*(s_i)$ 表示状态 $s_i$ 的价值函数，$A$ 表示动作集合，$S$ 表示状态集合，$R(s_i, a, s_j)$ 表示状态 $s_i$ 在执行动作 $a$ 后转移到状态 $s_j$ 的奖励。

#### 4.3 强化学习数学模型
强化学习的数学模型主要涉及策略评估和策略优化：

- **策略评估（Policy Evaluation）**：
  $$ V^*(s_i) = \sum_{s_j \in S} P(s_j|s_i, \pi) \cdot [R(s_i, \pi, s_j) + \gamma V^*(s_j)] $$
  其中，$V^*(s_i)$ 表示状态 $s_i$ 的价值函数，$\pi$ 表示策略，$\gamma$ 表示折扣因子。

- **策略优化（Policy Optimization）**：
  $$ \pi^* = \arg\max_{\pi} \sum_{s_i \in S} \pi(s_i) \cdot V^*(s_i) $$
  其中，$\pi^*$ 表示最优策略。

### 4. Mathematical Models and Formulas & Detailed Explanation & Examples
#### 4.1 Decision Tree Mathematical Model
The core of the decision tree lies in the mathematical model used to select features and partition nodes. Here are the main mathematical models of decision trees:

- **Information Gain**:
  $$ IG(D, A) = H(D) - \sum_{v\in A} \frac{|D_v|}{|D|} H(D_v) $$
  Where, $H(D)$ represents the entropy of the dataset $D$, $A$ represents the feature set, and $D_v$ represents the value of feature $v$.

- **Gini Impurity**:
  $$ GI(D, A) = 1 - \sum_{v\in A} \frac{|D_v|}{|D|}^2 $$
  Where, $A$ represents the feature set, and $D_v$ represents the value of feature $v$.

#### 4.2 Finite State Machine Mathematical Model
The mathematical model of the finite state machine mainly involves state transition probability and state value function:

- **State Transition Probability**:
  $$ P(s_i \rightarrow s_j) = \frac{\text{State } s_i \text{ to state } s_j \text{ transition count}}{\text{Total transition count of state } s_i} $$

- **State Value Function**:
  $$ V^*(s_i) = \max_{a \in A} \sum_{s_j \in S} P(s_j|s_i, a) \cdot R(s_i, a, s_j) $$
  Where, $V^*(s_i)$ represents the value function of state $s_i$, $A$ represents the action set, $S$ represents the state set, and $R(s_i, a, s_j)$ represents the reward of state $s_i$ after performing action $a$ and transitioning to state $s_j$.

#### 4.3 Reinforcement Learning Mathematical Model
The mathematical model of reinforcement learning mainly involves policy evaluation and policy optimization:

- **Policy Evaluation**:
  $$ V^*(s_i) = \sum_{s_j \in S} P(s_j|s_i, \pi) \cdot [R(s_i, \pi, s_j) + \gamma V^*(s_j)] $$
  Where, $V^*(s_i)$ represents the value function of state $s_i$, $\pi$ represents the policy, and $\gamma$ represents the discount factor.

- **Policy Optimization**:
  $$ \pi^* = \arg\max_{\pi} \sum_{s_i \in S} \pi(s_i) \cdot V^*(s_i) $$
  Where, $\pi^*$ represents the optimal policy.

```python
import numpy as np
from sklearn.tree import DecisionTreeClassifier
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split

# 加载数据集

# 训练模型
clf.fit(X_train, y_train)

# 预测
y_pred = clf.predict(X_test)

# 评估模型
from sklearn.metrics import accuracy_score
print("Accuracy:", accuracy_score(y_test, y_pred))
```
