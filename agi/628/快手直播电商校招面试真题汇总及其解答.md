### 3.1 最长公共子序列（Longest Common Subsequence, LCS）

#### 题目描述

给定两个字符串 `str1` 和 `str2`，找出它们的最长公共子序列。最长公共子序列是指两个字符串中出现的最长序列，该序列中的字符在两个字符串中出现的顺序一致。

#### 解题思路

可以使用动态规划的方法求解最长公共子序列。定义一个二维数组 `dp`，其中 `dp[i][j]` 表示 `str1` 的前 `i` 个字符和 `str2` 的前 `j` 个字符的最长公共子序列的长度。

#### 操作步骤

1. 初始化一个二维数组 `dp`，大小为 `(m+1)×(n+1)`，其中 `m` 和 `n` 分别为 `str1` 和 `str2` 的长度。初始化所有元素为 `0`。
2. 从 `dp[0][0]` 开始，遍历 `dp` 的所有元素。
   - 如果 `str1[i-1] == str2[j-1]`，则 `dp[i][j] = dp[i-1][j-1] + 1`。
   - 如果 `str1[i-1] != str2[j-1]`，则 `dp[i][j] = max(dp[i-1][j], dp[i][j-1])`。
3. 最终 `dp[m][n]` 即为最长公共子序列的长度。

#### 代码实现

```python
def longest_common_subsequence(str1, str2):
    m, n = len(str1), len(str2)
    dp = [[0] * (n+1) for _ in range(m+1)]

    for i in range(1, m+1):
        for j in range(1, n+1):
            if str1[i-1] == str2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])

    return dp[m][n]

# 示例
str1 = "ABCBDAB"
str2 = "BDCABC"
print(longest_common_subsequence(str1, str2))  # 输出：5
```

### 3.2 环形链表（Linked List Cycle）

#### 题目描述

给定一个链表，判断链表中是否存在环形链表。

#### 解题思路

可以使用哈希表的方法判断链表中是否存在环形链表。遍历链表，将每个节点加入哈希表中。如果再次遇到已经加入哈希表的节点，说明链表中存在环形链表。

#### 操作步骤

1. 初始化一个空哈希表 `hash_set`。
2. 遍历链表中的每个节点：
   - 如果当前节点已经在哈希表中，说明链表中存在环形链表，返回 `True`。
   - 否则，将当前节点加入哈希表中。
3. 如果遍历结束，说明链表中不存在环形链表，返回 `False`。

#### 代码实现

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def has_cycle(head):
    hash_set = set()

    while head:
        if head in hash_set:
            return True
        hash_set.add(head)
        head = head.next

    return False

# 示例
# 创建环形链表：1 -> 2 -> 3 -> 4 -> 5
# 环的入口为节点3
node1 = ListNode(1)
node2 = ListNode(2)
node3 = ListNode(3)
node4 = ListNode(4)
node5 = ListNode(5)
node1.next = node2
node2.next = node3
node3.next = node4
node4.next = node5
node5.next = node3

print(has_cycle(node1))  # 输出：True
```

### 3.3 合并两个有序链表（Merge Sorted Linked List）

#### 题目描述

给定两个有序链表 `l1` 和 `l2`，将它们合并成一个有序链表。

#### 解题思路

可以使用递归或迭代的方法合并两个有序链表。递归方法直接使用链表节点的指针进行合并，迭代方法使用一个哑节点来简化合并过程。

#### 操作步骤

1. 初始化一个哑节点 `dummy`，其值为 `None`，并设置当前节点 `current` 为 `dummy`。
2. 比较两个链表的头节点的值，将较小的节点链接到当前节点。
3. 移动当前节点和较小链表的头节点。
4. 重复步骤 2 和 3，直到其中一个链表为空。
5. 将非空链表的剩余部分链接到当前节点的 `next`。
6. 返回哑节点的 `next` 作为合并后的有序链表的头节点。

#### 代码实现

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def merge_sorted_lists(l1, l2):
    dummy = ListNode()
    current = dummy

    while l1 and l2:
        if l1.val < l2.val:
            current.next = l1
            l1 = l1.next
        else:
            current.next = l2
            l2 = l2.next
        current = current.next

    if l1:
        current.next = l1
    elif l2:
        current.next = l2

    return dummy.next

# 示例
# l1: 1 -> 3 -> 5
# l2: 2 -> 4 -> 6
node1 = ListNode(1)
node2 = ListNode(3)
node3 = ListNode(5)
node4 = ListNode(2)
node5 = ListNode(4)
node6 = ListNode(6)
node1.next = node2
node2.next = node3
node3.next = node5
node4.next = node5
node5.next = node6

merged_list = merge_sorted_lists(node1, node4)
while merged_list:
    print(merged_list.val, end=" -> ")
    merged_list = merged_list.next
# 输出：1 -> 2 -> 3 -> 4 -> 5 -> 6 ->
```

在技术面试中，数学模型和公式是解决算法题的重要工具。以下将详细讲解几个常用的数学模型和公式，并举例说明其在算法题中的应用。

### 4.1 动态规划（Dynamic Programming）

动态规划是一种解决最优化问题的方法，通过将问题分解为子问题并利用子问题的解来构建原问题的解。以下是一个常见的动态规划公式：

$$
dp[i][j] = 
\begin{cases}
0 & \text{if } i < 0 \text{ or } j < 0 \\
dp[i-1][j-1] + nums[i][j] & \text{if } i \geq 0 \text{ and } j \geq 0 \\
\min(dp[i-1][j], dp[i][j-1]) & \text{if } i \geq 0 \text{ and } j \geq 0 \text{ and } i = j \\
\end{cases}
$$

其中，`dp[i][j]` 表示从 `(i, j)` 到 `(m, n)` 的最长公共子序列的长度。

#### 示例

给定两个矩阵：

```
nums = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
```

使用动态规划求解从左上角 `(0, 0)` 到右下角 `(2, 2)` 的最长公共子序列。

1. 初始化 `dp` 矩阵为全零：
   $$
   dp = \begin{bmatrix}
   0 & 0 & 0 \\
   0 & 0 & 0 \\
   0 & 0 & 0
   \end{bmatrix}
   $$

2. 根据动态规划公式计算 `dp` 矩阵的每个元素：
   - `dp[0][0] = nums[0][0] = 1`
   - `dp[0][1] = dp[0][0] + nums[0][1] = 1 + 2 = 3`
   - `dp[0][2] = dp[0][1] + nums[0][2] = 3 + 3 = 6`
   - `dp[1][0] = dp[1][0] + nums[1][0] = 4`
   - `dp[1][1] = dp[1][0] + nums[1][1] = 4 + 5 = 9`
   - `dp[1][2] = dp[1][1] + nums[1][2] = 9 + 6 = 15`
   - `dp[2][0] = dp[2][0] + nums[2][0] = 7`
   - `dp[2][1] = dp[2][0] + nums[2][1] = 7 + 8 = 15`
   - `dp[2][2] = dp[2][1] + nums[2][2] = 15 + 9 = 24`

3. 最终 `dp[2][2]` 的值为 24，即从 `(0, 0)` 到 `(2, 2)` 的最长公共子序列的长度为 24。

### 4.2 贪心算法（Greedy Algorithm）

贪心算法是一种在每一步选择当前最优解的策略，以期望得到全局最优解。以下是一个常见的贪心算法公式：

$$
\text{max\_profit} = \sum_{i=1}^{n} (\text{current\_profit} - \text{cost}) \times \text{quantity}
$$

其中，`current_profit` 表示当前物品的利润，`cost` 表示当前物品的成本，`quantity` 表示当前物品的数量。

#### 示例

给定一个物品列表，其中每个物品的利润和成本如下：

```
items = [
    {'name': 'item1', 'profit': 10, 'cost': 5},
    {'name': 'item2', 'profit': 15, 'cost': 10},
    {'name': 'item3', 'profit': 20, 'cost': 15}
]
```

使用贪心算法计算所有物品的总利润。

1. 将物品列表按利润从高到低排序：
   ```
   sorted(items, key=lambda x: x['profit'], reverse=True)
   ```
   
2. 计算每个物品的利润贡献：
   - `item1` 的利润贡献：`(10 - 5) * 1 = 5`
   - `item2` 的利润贡献：`(15 - 10) * 2 = 10`
   - `item3` 的利润贡献：`(20 - 15) * 3 = 15`

3. 计算总利润：
   ```
   max_profit = 5 + 10 + 15 = 30
   ```

### 4.3 数学公式（Mathematical Formulas）

在算法题中，数学公式可以帮助我们更好地理解和解决问题。以下是一些常见的数学公式：

1. **勾股定理**：直角三角形斜边的平方等于两直角边的平方和。

   $$
   c^2 = a^2 + b^2
   $$

2. **均值不等式**：两个正数的算术平均数大于或等于它们的几何平均数。

   $$
   \frac{a + b}{2} \geq \sqrt{ab}
   $$

3. **二项式定理**：二项式的展开式。

   $$
   (a + b)^n = \sum_{k=0}^{n} \binom{n}{k} a^{n-k} b^k
   $$

通过以上数学模型和公式的讲解和举例，我们可以更好地理解它们在算法题中的应用，并提高解题效率。

#### 示例3：斐波那契数列（Fibonacci Sequence）

$$

F(n-1) + F(n-2) & \text{if } n > 1 \\
\end{cases}
$$

这个递推关系通过迭代计算得到斐波那契数列的第 n 个数。

### 5.2 源代码详细实现

以下是一个具体的编程实践题及其源代码实现：

#### 题目：计算两个整数的最大公约数

使用欧几里得算法计算两个整数的最大公约数。

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

# 示例
num1 = 24
num2 = 36
print(gcd(num1, num2))  # 输出：12
```

#### 解题思路

- `while b:`：使用 while 循环，当 `b` 不为0时继续执行。
- `a, b = b, a % b`：将 `b` 的值赋给 `a`，将 `a` 对 `b` 取余的结果赋给 `b`。
- `return a`：返回较大的数 `a`，即两个整数的最大公约数。

    n = len(arr)

        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
