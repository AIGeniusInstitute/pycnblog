## 4. 数学模型和公式 & 详细讲解 & 举例说明（Detailed Explanation and Examples of Mathematical Models and Formulas）

### 4.1 数学模型
在很多算法题和系统设计题中，会涉及到数学模型的应用。以下是一个常见的数学模型示例：

**题目**：设计一个二分查找算法。

**数学模型**：
1. 初始区间：[low, high]。
2. 中间值：mid = (low + high) / 2。
3. 比较目标值与中间值：
   - 如果目标值等于中间值，返回mid。
   - 如果目标值小于中间值，更新high = mid - 1。
   - 如果目标值大于中间值，更新low = mid + 1。

**举例说明**：
假设有一个有序数组[1, 3, 5, 7, 9]，要查找值为5的元素。

1. 初始区间：[0, 4]。
2. 第一次比较：mid = 2，目标值小于中间值，更新high = 2 - 1。
3. 第二次比较：mid = 1，目标值等于中间值，返回mid = 1。

### 4.2 数学公式
在某些系统设计题中，会涉及到数学公式的应用。以下是一个常见的数学公式示例：

**题目**：设计一个缓存淘汰算法。

**数学公式**：
1. 淘汰策略：最少使用（Least Recently Used，LRU）。
2. 假设缓存容量为C，当前缓存中有N个元素。
3. 每次访问元素时，如果元素在缓存中，则将其移动到缓存头部。
4. 如果访问的元素不在缓存中，且缓存已满，则需要淘汰缓存尾部元素。

**举例说明**：
假设缓存容量为3，当前缓存中有[1, 2, 3]，要访问元素4。

1. 元素4不在缓存中，缓存已满，需要淘汰缓存尾部元素3。
2. 将元素4添加到缓存头部，缓存变为[4, 2, 1]。

### 5.2 源代码详细实现
以下是一个简单的Java代码实例，用于实现二分查找算法：

```java
public class BinarySearch {
    public static int binarySearch(int[] array, int target) {
        int low = 0;
        int high = array.length - 1;
        
        while (low <= high) {
            int mid = (low + high) / 2;
            
            if (array[mid] == target) {
                return mid;
            } else if (array[mid] < target) {
                low = mid + 1;
            } else {
                high = mid - 1;
            }
        }
        
        return -1;
    }
    
    public static void main(String[] args) {
        int[] array = {1, 3, 5, 7, 9};
        int target = 5;
        int result = binarySearch(array, target);
        
        if (result == -1) {
            System.out.println("元素不在数组中");
        } else {
            System.out.println("元素在数组中的索引为：" + result);
        }
    }
}
```

### 5.3 代码解读与分析
这段代码实现了二分查找算法，主要分为以下几个步骤：

1. 初始化low和high指针，分别指向数组的起始和结束位置。
2. 进入while循环，当low <= high时，继续查找。
3. 计算中间值mid，并与目标值target进行比较。
4. 如果目标值等于中间值，返回mid。
5. 如果目标值小于中间值，更新high指针。
6. 如果目标值大于中间值，更新low指针。
7. 如果循环结束时仍未找到目标值，返回-1。
