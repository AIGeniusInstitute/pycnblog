### 4.2 公式推导过程

以下以行为监控模型为例，推导其中的关键公式。

假设行为监控模型的输入为行为序列$X=\{x_1, x_2, \cdots, x_n\}$，其中$x_i$表示第$i$个行为特征。模型的目标是为每个行为打分，用于判断是否为异常行为。假设模型的输出为行为得分$y=\{y_1, y_2, \cdots, y_n\}$，其中$y_i$表示第$i$个行为的得分。模型的训练目标是最小化预测误差，即：

$$
\min_{\theta} \sum_{i=1}^n (y_i - f(x_i; \theta))^2
$$

其中$f(x_i; \theta)$表示模型的预测函数，$\theta$为模型的参数。通过最小化均方误差，训练模型得到最优参数$\theta^*$。

    def __init__(self):

        self.conv1 = tf.keras.layers.Conv2D(32, (3, 3), activation='relu')
        self.conv2 = tf.keras.layers.Conv2D(64, (3, 3), activation='relu')
        self.flatten = tf.keras.layers.Flatten()
        self.dense1 = tf.keras.layers.Dense(64, activation='relu')
        self.dense2 = tf.keras.layers.Dense(1, activation='sigmoid')

    def call(self, inputs):
        x = self.conv1(inputs)
        x = self.conv2(x)
        x = self.flatten(x)
        x = self.dense1(x)
        x = self.dense2(x)
        return x

# 构建行为监控数据集
class BehaviorDataset(tf.keras.utils.Sequence):
    def __init__(self, X, y, batch_size=32):

        self.batch_size = batch_size

    def __len__(self):
        return len(self.y) // self.batch_size

        batch_x = self.X[idx * self.batch_size : (idx + 1) * self.batch_size]
