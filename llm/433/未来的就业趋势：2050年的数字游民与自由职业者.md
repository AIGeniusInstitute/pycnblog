### 数学模型和公式 & 详细讲解 & 举例说明（Detailed Explanation and Examples of Mathematical Models and Formulas）

#### 4.1 时间序列分析模型

时间序列分析是分析就业市场趋势的一种常用方法。时间序列数据是一系列按时间顺序排列的数值。以下是一个简单的时间序列分析模型：自回归积分滑动平均模型（ARIMA）。

- **ARIMA模型**：自回归积分滑动平均模型（Autoregressive Integrated Moving Average, ARIMA）是一个统计模型，用于分析时间序列数据。它结合了自回归模型（AR）、差分模型（I）和移动平均模型（MA）。

- **自回归模型（AR）**：自回归模型假设当前值可以通过过去值的线性组合来预测。公式如下：

  $$y_t = c + \phi_1 y_{t-1} + \phi_2 y_{t-2} + \ldots + \phi_p y_{t-p} + \epsilon_t$$

  其中，$y_t$是时间序列的当前值，$\phi_1, \phi_2, \ldots, \phi_p$是自回归系数，$c$是常数项，$\epsilon_t$是误差项。

- **移动平均模型（MA）**：移动平均模型假设当前值可以通过过去值的加权平均来预测。公式如下：

  $$y_t = c + \theta_1 \epsilon_{t-1} + \theta_2 \epsilon_{t-2} + \ldots + \theta_q \epsilon_{t-q}$$

  其中，$\theta_1, \theta_2, \ldots, \theta_q$是移动平均系数，$\epsilon_t$是误差项。

- **差分模型（I）**：差分模型用于消除时间序列数据中的趋势和季节性。差分可以通过以下公式实现：

  $$y_t = y_{t-1} - y_{t-2}$$

  或

  $$y_t = (1 - B)^d y_t$$

  其中，$B$是滞后算子，$d$是差分的阶数。

#### 4.2 回归分析模型

- **Autoregressive Model (AR)**: The autoregressive model assumes that the current value can be predicted by a linear combination of past values. The formula is as follows:

  $$y_t = c + \phi_1 y_{t-1} + \phi_2 y_{t-2} + \ldots + \phi_p y_{t-p} + \epsilon_t$$

  Where $y_t$ is the current value of the time series, $\phi_1, \phi_2, \ldots, \phi_p$ are the autoregressive coefficients, $c$ is the constant term, and $\epsilon_t$ is the error term.

- **Moving Average Model (MA)**: The moving average model assumes that the current value can be predicted by the weighted average of past values. The formula is as follows:

  $$y_t = c + \theta_1 \epsilon_{t-1} + \theta_2 \epsilon_{t-2} + \ldots + \theta_q \epsilon_{t-q}$$

  Where $\theta_1, \theta_2, \ldots, \theta_q$ are the moving average coefficients, and $\epsilon_t$ is the error term.

- **Differencing Model (I)**: The differencing model is used to remove trends and seasonality from time series data. Differencing can be achieved through the following formula:

  $$y_t = y_{t-1} - y_{t-2}$$

  Or

  $$y_t = (1 - B)^d y_t$$

  Where $B$ is the lag operator, and $d$ is the order of differencing.

import numpy as np
import matplotlib.pyplot as plt
import statsmodels.api as sm
from statsmodels.tsa.stattools import adfuller

# 读取数据
data = pd.read_csv('employment_data.csv')

# 检查数据
print(data.head())

    dftest = adfuller(timeseries, autolag='AIC')
    dfoutput = pd.Series(dftest[0:4], index=dftest.names)
    return dfoutput

print("Results of ADF Test:")
print(test_stationarity(data['employment_rate']))

# 进行差分
data['employment_rate_diff'] = data['employment_rate'].diff().dropna()

# 检查差分后的序列是否稳定
print("Results of ADF Test for Differenced Series:")
print(test_stationarity(data['employment_rate_diff']))

# 建立ARIMA模型
model = sm.ARIMA(data['employment_rate'], order=(1, 1, 1))
model_fit = model.fit()

# 预测未来10个时间点的就业率
predictions = model_fit.forecast(steps=10)

# 可视化
plt.plot(data['employment_rate'], label='Actual')
plt.plot(predictions, label='Predicted')
plt.legend()
plt.show()

#### 4.1 Time Series Analysis Models

- **ARIMA Model**: The ARIMA model, which stands for Autoregressive Integrated Moving Average, is a widely used time series forecasting model. It consists of three main components: autoregression (AR), differencing (I), and moving average (MA). The general formula for ARIMA(p, d, q) is:

  $$Y_t = c + \phi_1 Y_{t-1} + \phi_2 Y_{t-2} + \ldots + \phi_p Y_{t-p} + \theta_1 \varepsilon_{t-1} + \theta_2 \varepsilon_{t-2} + \ldots + \theta_q \varepsilon_{t-q} + \varepsilon_t$$

  where $Y_t$ is the time series value at time $t$, $c$ is a constant term, $\phi_1, \phi_2, \ldots, \phi_p$ are the autoregressive coefficients, $\theta_1, \theta_2, \ldots, \theta_q$ are the moving average coefficients, and $\varepsilon_t$ is the white noise error term.

  - **Autoregression (AR)**: AR models predict the current value based on the past values. The formula for AR(p) is:

    $$Y_t = \phi_1 Y_{t-1} + \phi_2 Y_{t-2} + \ldots + \phi_p Y_{t-p} + \varepsilon_t$$

  - **Moving Average (MA)**: MA models predict the current value based on the past errors. The formula for MA(q) is:

    $$Y_t = \theta_1 \varepsilon_{t-1} + \theta_2 \varepsilon_{t-2} + \ldots + \theta_q \varepsilon_{t-q} + \varepsilon_t$$

  - **Differencing (I)**: Differencing is used to stabilize the time series data by removing trends and seasonality. The first difference is calculated as:

    $$\Delta Y_t = Y_t - Y_{t-1}$$

    Higher-order differences can be calculated recursively:

    $$\Delta^2 Y_t = \Delta Y_t - \Delta Y_{t-1}$$

  $$Y = \beta_0 + \beta_1 X + \varepsilon$$

  where $Y$ is the dependent variable, $X$ is the independent variable, $\beta_0$ is the intercept, $\beta_1$ is the slope, and $\varepsilon$ is the error term.

- **Multiple Linear Regression**: Multiple linear regression extends simple linear regression to include multiple independent variables. The formula is:

  $$Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \ldots + \beta_n X_n + \varepsilon$$

  $$X = \alpha_1 P_1 + \alpha_2 P_2 + \ldots + \alpha_p P_p + \varepsilon$$

  where $P_1, P_2, \ldots, P_p$ are the principal components.
