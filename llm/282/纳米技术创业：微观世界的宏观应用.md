#### 5.2 源代码详细实现

```python
import numpy as np
import matplotlib.pyplot as plt

# 设置尺寸分布参数
mu = 10  # 平均尺寸
sigma = 2  # 标准差

# 生成尺寸分布数据
d = np.linspace(mu - 3*sigma, mu + 3*sigma, 100)
f = 1/(sigma * np.sqrt(2 * np.pi)) * np.exp(-( (d - mu)**2 ) / (2 * sigma**2))

# 绘制尺寸分布曲线
plt.plot(d, f)
plt.xlabel('尺寸 (纳米)')
plt.ylabel('尺寸分布')
plt.title('纳米材料尺寸分布')
plt.show()
```
