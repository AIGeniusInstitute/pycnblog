### 4.3 数学模型举例说明

#### 线性规划举例

假设我们要最小化成本 $C = [1, 2]$，同时满足 $Ax \leq b$，其中 $A = [1, 0; 0, 1]$，$b = [2, 2]$。

- **目标函数**：
  $$ \text{minimize} \ C^T x = 1 \cdot x_1 + 2 \cdot x_2 $$
- **约束条件**：
  $$ x_1 + x_2 \leq 2 $$
  $$ x_1 \geq 0 $$
  $$ x_2 \geq 0 $$

通过求解线性规划问题，我们得到最优解 $x = [1, 1]$，此时目标函数取最小值 $C^T x = 3$。

#### 决策树举例

假设我们要对数据集进行分类，其中有两个特征 $x_1$ 和 $x_2$，各自的概率分布如下：

- $p(x_1 = 0) = 0.6, p(x_1 = 1) = 0.4$
- $p(x_2 = 0) = 0.5, p(x_2 = 1) = 0.5$

熵计算如下：

$$ \text{Entropy}(S) = - (0.6 \cdot \log_2 0.6 + 0.4 \cdot \log_2 0.4) - (0.5 \cdot \log_2 0.5 + 0.5 \cdot \log_2 0.5) $$
$$ \text{Entropy}(S) = - (0.6 \cdot (-0.737) + 0.4 \cdot (-1.38)) - (0.5 \cdot (-1) + 0.5 \cdot (-1)) $$
$$ \text{Entropy}(S) = 0.366 + 0.552 + 0.5 $$
$$ \text{Entropy}(S) = 1.418 $$

#### 聚类算法举例

假设我们使用 K-Means 算法对数据集进行聚类，数据集包含三个特征 $x_1, x_2, x_3$，聚类中心分别为 $\mu_1, \mu_2, \mu_3$。

- **SSE** 计算：
  $$ \text{SSE} = (x_1 - \mu_1)^2 + (x_2 - \mu_2)^2 + (x_3 - \mu_3)^2 $$
- 假设数据集为：
  $$ \begin{aligned}
  &x_1 = [2, 3, 4], \quad \mu_1 = [2.5, 3, 3.5] \\
  &x_2 = [5, 6, 7], \quad \mu_2 = [5.5, 6, 6.5] \\
  &x_3 = [8, 9, 10], \quad \mu_3 = [8.5, 9, 9.5] \\
  \end{aligned} $$

计算 SSE 如下：

$$ \text{SSE} = (2 - 2.5)^2 + (3 - 3)^2 + (4 - 3.5)^2 + (5 - 5.5)^2 + (6 - 6)^2 + (7 - 6.5)^2 + (8 - 8.5)^2 + (9 - 9)^2 + (10 - 9.5)^2 $$
$$ \text{SSE} = 0.25 + 0 + 0.25 + 0.25 + 0 + 0.25 + 0.25 + 0 + 0.25 $$
$$ \text{SSE} = 1.5 $$

#### 主成分分析举例

假设我们有一个三维数据集，特征协方差矩阵为 $L$，其特征值为 $\lambda_1, \lambda_2, \lambda_3$。

- **特征值**：
  $$ \lambda_1 = 2, \quad \lambda_2 = 1, \quad \lambda_3 = 0.5 $$
- **方差**：
  $$ \text{Variance}(Z) = \lambda_1 = 2 $$

```python
# 题目：给定一个整数数组，找出所有出现次数超过数组长度一半的元素

def majority_element(nums):
    count = 0
    candidate = None

    for num in nums:
        if count == 0:
            candidate = num
        count += (1 if num == candidate else -1)

    return candidate

# 测试代码
nums = [2, 2, 1, 1, 1, 2, 2]
print(majority_element(nums))  # 输出：2
```

### 5.3 代码解读与分析

- **核心算法**： Boyer-Moore 投票算法，时间复杂度为 O(n)，空间复杂度为 O(1)。
- **代码实现**：通过遍历数组，维护当前候选数和计数器。当计数器为0时，更新候选数；否则，根据当前数是否与候选数相等来更新计数器。

- **选择排序**：每次从未排序的元素中找到最小（或最大）的元素，将其交换到已排序的序列末尾。

#### 动态规划

动态规划是一种用于求解最优化问题的算法方法，通常用于解决背包问题、最短路径问题等。动态规划的基本思想是：

1. **定义状态**：将问题分解为多个子问题，定义每个子问题的状态。
2. **状态转移方程**：找到状态之间的转移关系，建立状态转移方程。
3. **初始化**：根据初始条件初始化状态。
4. **计算最优解**：从初始状态开始，逐步计算出所有状态的最优解。

#### 举例：寻找最长公共子序列

**题目描述**：给定两个字符串 `s1` 和 `s2`，找到它们的最长公共子序列。

**算法**：动态规划

**步骤**：

1. **定义状态**：设 `dp[i][j]` 表示 `s1` 的前 `i` 个字符和 `s2` 的前 `j` 个字符的最长公共子序列长度。
2. **状态转移方程**：
   $$ dp[i][j] = \begin{cases} 
   dp[i-1][j-1] + 1, & \text{如果 } s1[i-1] = s2[j-1]; \\
   \max(dp[i-1][j], dp[i][j-1]), & \text{否则};
   \end{cases} $$
3. **初始化**：设 `dp[0][j] = dp[i][0] = 0`。
4. **计算最优解**：根据状态转移方程计算 `dp[m][n]` 的值，即为最长公共子序列的长度。
5. **回溯**：从 `dp[m][n]` 开始，回溯找到最长公共子序列。

```python
def longest_common_subsequence(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    # 回溯得到最长公共子序列
    result = []
    i, j = m, n
    while i > 0 and j > 0:
        if s1[i - 1] == s2[j - 1]:
            result.append(s1[i - 1])
            i -= 1
            j -= 1
        elif dp[i - 1][j] > dp[i][j - 1]:
            i -= 1
        else:
            j -= 1

    return result[::-1]

# 测试代码
s1 = "ABCBDAB"
s2 = "BDCABC"
print(longest_common_subsequence(s1, s2))  # 输出：['B', 'D', 'A', 'B', 'C']
```

**Algorithm**: Dynamic Programming

**Steps**:

1. **Define the State**: Let `dp[i][j]` represent the length of the longest common subsequence of the first `i` characters of `s1` and the first `j` characters of `s2`.
2. **State Transition Equation**:

5. **Backtracking**: Starting from `dp[m][n]`, backtrack to find the longest common subsequence.

```python

    result = []

        if s1[i - 1] == s2[j - 1]:

            i -= 1
            j -= 1

            i -= 1
        else:
            j -= 1
```

### 4.1 数学模型和公式的介绍

在解决编程题和系统设计题时，数学模型和公式起到了至关重要的作用。它们帮助我们理解和分析问题，提供解决方法的数学依据。以下是一些常见的数学模型和公式：

#### 线性规划

线性规划是一种数学优化方法，用于求解资源分配问题。其标准形式如下：

$$
\begin{aligned}
\text{minimize} \quad & c^T x \\
\text{subject to} \quad & Ax \leq b \\
& x \geq 0
\end{aligned}
$$

其中，$c$ 是系数向量，$x$ 是变量向量，$A$ 是约束矩阵，$b$ 是常数向量。

#### 决策树

决策树是一种用于分类和回归问题的图形化模型。其熵的计算公式为：

$$
\text{Entropy}(S) = -\sum_{i=1}^{n} p_i \log_2 p_i
$$

其中，$p_i$ 是各个类别的概率。

#### 聚类算法

聚类算法用于将数据集划分为多个群组。K-Means 算法是一种常用的聚类算法，其平方误差和（SSE）的计算公式为：

$$
\text{SSE} = \sum_{i=1}^{n} (x_i - \mu)^2
$$

其中，$x_i$ 是数据点，$\mu$ 是聚类中心。

#### 主成分分析

主成分分析是一种降维方法，用于从原始数据中提取最重要的特征。其特征值的计算公式为：

$$
\text{Variance}(Z) = \text{Eigenvalue}(L)
$$

其中，$L$ 是协方差矩阵。

**Example**: You have 100 hours of work time and need to complete the following tasks:

- Programming: Each project requires 2 hours and yields a profit of 50,000 yuan.
- Learning: Each project requires 3 hours and yields a profit of 30,000 yuan.

How should you allocate your time to maximize profits?

**Solution**:

1. **Define Variables**: Let $x$ be the number of programming projects and $y$ be the number of learning projects.
2. **Establish the Objective Function**: Maximize profit $f(x, y) = 5x + 3y$.
3. **Establish Constraints**:
   $$
   \begin{aligned}
   2x + 3y &\leq 100 \\
   x &\geq 0 \\
   y &\geq 0
   \end{aligned}
   $$
4. **Solve the Linear Programming Problem**: Use the simplex method or other algorithms to solve.

The solution is $x = 25, y = 15$, resulting in the maximum profit of 200,000 yuan.

```python
import smtplib
from email.mime.text import MIMEText
from email.header import Header

def send_email(sender, receiver, subject, content):
    # 邮件发送服务器地址和端口
    smtp_server = 'smtp.example.com'
    smtp_port = 587

    # 发送者和接收者的邮箱地址和密码
    sender_email = 'sender@example.com'
    receiver_email = 'receiver@example.com'
    password = 'your_password'

    # 创建邮件对象
    message = MIMEText(content, 'plain', 'utf-8')
    message['From'] = Header(sender, 'UTF-8')
    message['To'] = Header(receiver, 'UTF-8')
    message['Subject'] = Header(subject, 'UTF-8')

    # 发送邮件
    try:
        server = smtplib.SMTP(smtp_server, smtp_port)
        server.starttls()
        server.login(sender_email, password)
        server.sendmail(sender_email, receiver_email, message.as_string())
        server.quit()
        print("邮件发送成功！")
    except smtplib.SMTPException as e:
        print("邮件发送失败！原因：", e)

def receive_email(sender, password):
    # 邮件接收服务器地址和端口
    smtp_server = 'smtp.example.com'
    smtp_port = 993

    # 发送者的邮箱地址和密码
    receiver_email = 'receiver@example.com'
    password = 'your_password'

    # 创建邮件对象
    message = MIMEText()
    message['From'] = Header(sender, 'UTF-8')
    message['To'] = Header(receiver_email, 'UTF-8')

    # 接收邮件
    try:
        server = smtplib.SMTP_SSL(smtp_server, smtp_port)
        server.login(sender_email, password)
        messages = server.fetch(receiver_email, "[Gmail]/All Mail")[1]
        for response in messages:
            if response[0] == '1':
                print("邮件主题：", response[2][1])
                print("邮件内容：", response[2][2])
                print("\n")
    except smtplib.SMTPException as e:
        print("邮件接收失败！原因：", e)

# 测试代码
send_email('sender@example.com', 'receiver@example.com', '测试邮件', '这是一封测试邮件。')
receive_email('receiver@example.com', 'your_password')
```

- **接收邮件**：同样使用`smtplib`库，通过`server.login()`登录邮箱，然后调用`server.fetch()`获取所有邮件，遍历邮件列表，打印邮件的主题和内容。
