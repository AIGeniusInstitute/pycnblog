```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
```

```python
def binary_search(arr, target):
    low = 0
    high = len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1
```

#### 4.3.3 图算法

Dijkstra算法的实现代码如下：

```python
def dijkstra(graph, start):
    dist = [float('inf')] * len(graph)
    dist[start] = 0
    visited = [False] * len(graph)
    for _ in range(len(graph)):
        min_dist = float('inf')
        min_index = -1
        for i in range(len(graph)):
            if not visited[i] and dist[i] < min_dist:
                min_dist = dist[i]
                min_index = i
        visited[min_index] = True
        for i in range(len(graph)):
            if not visited[i]:
                dist[i] = min(dist[i], min_dist + graph[min_index][i])
    return dist
```

### 核心算法原理 & 具体操作步骤

在算法面试中，理解核心算法原理和具体操作步骤至关重要。以下将介绍一些常见的核心算法及其操作步骤：

#### 动态规划

1. **定义状态**：将问题分解为子问题，定义每个子问题的状态。
2. **确定状态转移方程**：找出子问题之间的关联，建立状态转移方程。
3. **初始化边界条件**：确定初始状态和边界条件。
4. **递推求解**：根据状态转移方程，递推求解每个状态。
5. **输出结果**：根据最后的状态，输出最终结果。

#### 贪心算法

1. **选择局部最优解**：在每一步选择中都采取当前最优解的策略。
2. **验证全局最优性**：通过验证，证明最终结果为全局最优解。

#### 图算法

1. **深度优先搜索（DFS）**：遍历图中的每个节点，找出所有可能的路径。
2. **广度优先搜索（BFS）**：从起始节点开始，按照层次遍历图中的每个节点。
3. **最短路径算法**：如Dijkstra算法、Floyd算法等，用于求解图中两点之间的最短路径。
4. **最小生成树算法**：如Prim算法、Kruskal算法等，用于求解图中的最小生成树。

```python
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]

# 测试代码
arr = [64, 25, 12, 22, 11]
bubble_sort(arr)
print("排序后的数组：", arr)
```

#### 代码解读与分析

1. **函数定义**：`bubble_sort`函数用于实现冒泡排序算法。
2. **外层循环**：遍历数组中的每个元素，将其与后面的元素进行比较，找到最小元素的下标。
3. **内层循环**：遍历数组中剩余的元素，找到最小元素的下标。
4. **交换元素**：将找到的最小元素与当前元素进行交换。
5. **测试代码**：创建一个测试数组，调用`bubble_sort`函数进行排序，并打印排序后的数组。

### 附录：常见问题与解答

#### 问题1

**问题**：什么是动态规划？

**解答**：动态规划是一种将复杂问题分解为子问题，并利用子问题的解来求解原问题的算法设计技术。它通常用于解决具有重叠子问题和最优子结构性质的问题。

#### 问题2

**问题**：什么是贪心算法？

**解答**：贪心算法是一种在每一步选择中都采取当前最优解的策略的算法。它通过选择局部最优解，希望最终能够得到全局最优解。

#### 问题3

**问题**：什么是图算法？

**解答**：图算法是解决与图相关问题的算法集合，包括最短路径算法、最小生成树算法、图着色算法等。图算法广泛应用于网络路由、社交网络分析等领域。
